# Website Theme: Minimal, Premium, Technical

The website is a clean, recruiter-friendly portfolio for an AI & Research Engineer. It should feel like a high-end startup / product site — think Apple, Linear, OpenAI, Vercel, and Anthropic — not a personal blog, student portfolio, or template. The tone is minimal, premium, and technical, with large typography and generous whitespace so everything breathes.

## Design language

- **Dark, restrained, and glassy.** Near-black background with subtle radial accent glows and a faint technical grid that fades toward the bottom.
- **Whitespace first.** Large sections, clear hierarchy, plenty of breathing room.
- **Rounded cards, hairline borders, soft shadows.** Surfaces are translucent white over the dark background.
- **Sparing accent.** A cyan→purple gradient is used only on key words, primary buttons, and active states.

## Color palette

- Background: `#07080A` / `#0B0D11`
- Cards / surfaces: `rgba(255,255,255,0.05)` (stronger on hover)
- Borders: `rgba(255,255,255,0.10)`
- Text: white `#F4F7FB`, soft `#CCD5DF`, muted slate `#9AA6B2`
- Accent: cyan `#72E4FF` and purple `#A78BFA`

Avoid: bright rainbow gradients, orange, earthy/nature tones, and green as a primary color.

## Typography

Inter, with large headings and tight letter-spacing.

- Hero: ~72px (fluid)
- Section titles: ~48px
- Card titles: ~24px
- Body: 16–18px

## Layout

Sticky minimal navbar → Hero (headline + profile snapshot card) → About → Work (timeline) → Projects → Research → Skills → Milestones → Contact → Footer. Each project card leads with what it is, why it matters, the tech stack, and measurable impact.

## Motion

Minimal only: fade-in on scroll, hover elevation, and subtle transitions. No parallax, floating elements, bouncing, rotation, or heavy scroll animations. All motion respects `prefers-reduced-motion`.

## Goal

Communicate craftsmanship, technical depth, and professionalism. The result should look suitable for an AI Engineer applying to OpenAI, Anthropic, NVIDIA, Tesla AI, Scale AI, or top AI startups.
